# Spatial Triple-Correlation Spectroscopy (S3CS) 

The provided code for spatial triple correlation microscopy is designed for researchers seeking to quantify interaction among protein subunits within individual complexes and to assess associations between distinct protein assemblies in live cells. The code implements high-order correlation analysis to detect the formation and dynamics of heterotrimeric protein complexes and to distinguish between reversible and stable interaction processes across subcellular compartments. The intent is to enable the spatial mapping of heterotrimer mobility within intracellular environments and allow for the accurate identification of fully assembled heterotrimeric complexes relative to substoichiometric intermediates. Improvements and open-source data will be continuously included to facilitate accessibility and adoption of the method.
